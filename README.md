# 2048 + Nuxt 3

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install --shamefully-hoist
```

## Development Server

Start the development server on http://localhost:3000

```bash
npm run dev

# or

yarn dev
```


Game can be accessed on http://localhost:3000/game
